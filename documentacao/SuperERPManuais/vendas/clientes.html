Clientes

Clientes são empresas ou pessoas para quem você vende seus produtos e serviços, emite nota fiscal ou envia boletos de pagamento. Neste espaço você poderá informar os dados pessoais, endereço e contatos do seu cliente. Se seu cliente for uma empresa, informe o CNPJ e encontre rapidamente as informações sobre ele.

Para acessar as funções disponíveis de clientes, clique em “Vendas” e depois “Clientes”.

Dentro de clientes é possível:

Listar o Cadastro de Clientes

A tela inicial mostra todos os clientes já cadastrados.

Listagem de Clientes

Ações possíveis:

  • Cadastro de um novo Cliente

  • Para a edição do registro basta clicar no registro do cliente

  • Para Excluir um registro.

  • Botão de Navegação por página

  • Pesquisar pelo nome do cliente

  • Definir a quantidade de registros por página

Ações possíveis:

  • Pessoa física

    Para realizar o cadastro de Pessoa Física as informações obrigatórias são:

    • Nome do Cliente
    • Tipo Pessoa
    • CPF
    • RG
    • E-mail
    • Telefone
    • Cargo
    • Endereço
    • Número
    • Complemento
    • CEP
    • Bairro
    • Cidade
